### 1.1 비용 표시 절대 금지 정책 (2026-05-06 갱신)

ClassAuto는 **구독제(월 정액)** 입니다. 교수자에게 비용($/₩/토큰) 노출은 다음 부작용을 일으킵니다:

- "이거 비싸지 않나?" 불안 유발 → 자유로운 사용 저해
- "한 번 만들 때마다 얼마 나가나?" 신경 쓰게 함 → 작업 흐름 끊김
- 구독제의 본질(마음껏 써도 됨)과 모순

대신 표시할 것:
- **편수 한도**: "18 / 20편" (Pro 플랜)
- **학생 수 한도**: "47 / 150명"
- **진행 정보**: "슬라이드 5 / 8"
- **예상 시간**: "남은 시간 4분 52초"

원가 모니터링은 어흥 교수님(관리자) 전용 admin 페이지에서만 (별도 화면).

## 3. 교수자 첫 사용 온보딩

### 3.1 환영 모달 (첫 로그인 시)
- "안녕하세요, ○○○ 교수님"
- 5단계 가이드 미리보기
- "지금 시작" / "둘러보기"

### 3.2 빈 상태 디자인 (Empty State)
- 통계 카드는 모두 "—" (의미 있는 placeholder)
- 메인 영역에 큰 "첫 강의 영상 만들기" CTA
- 샘플 데이터로 미리보기 가능한 토글 ("샘플 강의로 둘러보기")

### 3.3 첫 영상 제작 가이드 체크리스트
사이드 패널에:
- [ ] PPT 업로드
- [ ] 스크립트 검토
- [ ] AI 아바타 선택
- [ ] 영상 생성
- [ ] 학생에게 공유

각 단계 완료 시 ✓로 변경, 100% 완료 시 축하 모달.

### 3.4 학과/강의 정보 입력
- 소속 학교, 학과
- 가르치는 과목명
- 학기 시작일
- 학생 수 (선택적 — 정확한 한도 안내용)

이 데이터가 있어야 대시보드의 "이번 학기 ○주차" 같은 컨텍스트가 작동.

### 4.1 상단 영역
- **컨텍스트 바**: "안녕하세요, 하두진 교수님 · 2026 봄학기 · 중국어문법의 이해" + 학기 전환 셀렉트 + 검색
- **인사 카드**: "이번 주 새 질문 12개, 시청 미완료 학습자 7명" 한 줄 요약

| 카드 | 표시 내용 | 색상 의미 |
|---|---|---|
| 시청 완료율 | 78% (전주 대비 ▲4%p) | 녹색 화살표 |
| 평균 정답률 | 82% | 무채색 |

| 이번 달 영상 | 8 / 20편 (Pro) | 진행 바 |
| ~~누적 사용 비용~~ | **사용한 영상 편수 진행** | 진행 바 (편수 단위) |

⚠️ **6번째 카드 변경**: 기존 "누적 사용 비용 $46.20"을 비용 표시 제거 정책에 따라 다음으로 교체:
- **이번 달 학습자 활동**: "342시간 누적 시청" 또는
- **다음 영상 제작 예정**: 캘린더 마감일 카운트다운

### 4.3 메인 차트 (좌측 2/3)
- **강의별 시청 추이** — 라인 차트(주차별), 강의 3~5개 토글
- ElevenAgents Success Rate 차트 톤 (얇은 라인, 무채색 + 그린 한 포인트)

### 4.4 우측 "주의 필요" 위젯 (1/3)
- **답변 대기 질문** (최신 5건) → 클릭 시 Q&A 인박스
- **시청 부진 학습자** (3일 이상 미시청, 5명) → 알림 발송
- **자주 멈춘 구간** (가장 많이 일시정지/되감기된 슬라이드 Top 3) → 다음 강의 보강 추천

### 4.5 하단
**최근 강의 영상 그리드** (썸네일 카드 4개) — 썸네일 + 강의명 + 시청 완료율 + Q&A 수 + 메뉴.

### 4.6 동적 요소
- 통계 카드 카운트업 애니메이션 (페이지 진입 시 1회)
- 변화율 옆 sparkline (지난 7일 추이)
- 카드 hover 시 sparkline 활성화

### 5.1 5단계 흐름 (2026-05-06 프로토타입 검증 완료)

| # | 화면 | 톤 | 상태 |

### 5.2 01 — PPT 업로드 모달

- "새 영상 만들기" 버튼 클릭 시 모달 등장

- 자동 분석 단계 표시:
  - ✓ 슬라이드 추출 완료 (8장)
  - ✓ 발표자 노트 추출 완료
  - ✓ AI 스크립트 생성 중...
  - ✓ 강의 분석 완료
- "마법사 시작하기 →" 버튼으로 02 진입

### 5.3 02 — 3단 마법사 메인

3단 레이아웃 (1280px 이상 데스크톱):
- **좌측 슬라이드 패널** (240px): 8장 썸네일 + 상태 아이콘 (●⚠○)
- **중앙 작업 영역** (가변): 미리보기 + 스크립트 검토 (diff 통합)
- **우측 설정 패널** (320px): 아바타·음성·강의 설정·Q&A 범위 아코디언

#### 5.3.1 중앙 스크립트 검토 — Diff 편집 모드 ⭐

뷰 토글 4개: [원본만] [AI 다듬은 것만] [Diff 보기 ⭐] [수동 편집]

**Diff 보기 — 통합 인라인 패턴**:
- 추가된 단어: 골드 highlight (`rgba(255, 182, 39, 0.15)` + `#B88308` 텍스트)
- 삭제된 단어: 회색 + 취소선 (`rgba(0,0,0,0.04)` + `#888`)
- 변경된 단어: 보라 highlight (`rgba(167, 139, 250, 0.10)` + `#6366F1`)

변경 통계: "● 47 단어 추가  ● 3 단어 삭제  ● 8 단어 변경"
변경 네비게이션: [↑ 이전] [↓ 다음] [3/12]

채택 옵션:
- [✓ 모두 채택] / [✗ 모두 거부] / [✎ 수동 편집] / [⊕ 개별 채택]
- 개별 채택 모드: 각 변경마다 작은 ✓✗ 버튼

#### 5.3.2 반응형 정책 (2026-05-06 확정)

데스크톱 앱 패턴 적용:
- 전체 컨테이너: `height: 100vh; overflow: hidden`
- 좌·중·우 영역 각자 `overflow-y: auto` (독립 스크롤)
- 중앙 미리보기: `max-height: 40vh` (콘텐츠 무한 늘어남 방지)

브레이크포인트 3개로 단순화:
- **1280px 이상**: 3단 유지 (240 / flex:1 / 320)
- **768~1280px**: 2단 (좌측 패널 햄버거 메뉴로 숨김)
- **768px 미만**: 1단 + 하단 시트

### 5.5 04 — 영상 완성 + 공유 패널 + QR 코드 ⭐


각 채널 클릭 시 모달 (mailto / KakaoLink / Twitter intent / clipboard / SMS).

#### 5.5.4 QR 코드 모달 ⭐ — ClassAuto의 차별점

QR 카드 클릭 시 모달 (max-width: 540px):

- 헤딩: "강의실에 QR 코드를 띄우세요"
- 큰 QR 코드 (320×320px)
  - 중앙에 ClassAuto 로고
  - 골드 / 다크 토글 (강의실 슬라이드용 / 인쇄용)
- 다운로드 옵션 3개:
  - 📥 PNG 다운로드 (고해상도)
  - 📥 PPT 슬라이드 템플릿 (.pptx에 QR 박힌 상태)
  - 📥 인쇄용 A4 PDF
- 사용 시나리오 3카드:
  - 🏫 강의실 슬라이드 마지막 페이지
  - 📋 시험·과제 인쇄물에 삽입
  - 📌 게시판·복도 포스터 부착

**QR 활용 시나리오의 의미**: 한국 대학 강의실 환경 최적화. 다른 글로벌 EdTech가 신경 안 쓰는 디테일.

#### 5.5.5 공유 설정 아코디언 (접힌 상태)

## 6. /inbox (Q&A 인박스)

Gmail 스타일 3단:

### 6.1 좌측: 강의별 필터
- 강의 목록 + 안 읽음 카운트
- "전체" 필터

### 6.2 중앙: 질문 리스트
3개 탭:
- AI 자동응답 (참고용 모니터링)
- 교수자 응답 필요 (액션 아이템)
- 범위 외 거부 (학생이 "교수님께 전달"한 것)

각 항목: 학생명·강의명·시간 + 질문 미리보기.

### 6.3 우측: 선택한 질문의 컨텍스트
- 어느 영상 어느 시점에서 질문했는가
- 학생이 본 슬라이드 미리보기
- AI 답변 초안
- **유사 질문 클러스터링 (Pro)**: "이 질문은 다른 학생 12명도 비슷하게 물었습니다 → 다음 강의 보강 추천"

## 7. /analytics (분석 리포트)

### 7.1 재생 구간 히트맵 (Pro)
- 영상 타임라인 위에 색 농도 (어느 구간을 반복 재생/건너뛰기)
- 영상별로 분리, 강의 단위 합산도 가능

### 7.2 취약점 히트맵 (Pro)
- 강의 챕터 × 학습자 매트릭스
- 정답률 낮은 셀이 빨갛게
- 행렬 정렬: 학습자별 / 챕터별 / 정답률 순
- "이 챕터를 보강하시겠습니까?" 액션

### 7.3 정답률·Q&A 트렌드 (Basic+)
- 강의별 정답률 추이
- Q&A 트렌드 (자주 묻는 토픽)
- ~~월별 비용 사용 추이~~ → 월별 **편수 사용** 추이 (비용 제거)

### 7.4 익명 반응 집계 (Basic+)
- 영상별 4가지 이모지 반응 비율
- 시점별 반응 분포

### 7.5 딴짓 포착 리포트 (Basic+)
- 의심 패턴 학습자 목록
- 시청 시간 < 영상 길이 30% 학습자
- 매크로 의심 행동 (가드레일 4차 데이터)

### 7.6 대면 수업 브리핑 자동 (Pro) ⭐ 어흥 교수님 강조 기능
- 영상 시청 데이터 기반 자동 생성
- "지난 영상에서 학생들이 가장 어려워한 부분: ○○○"
- "다음 대면 수업에서 다룰 주제 추천: △△△"
- ClassAuto의 핵심 가치 = 학습 데이터 → 대면 수업 개선
