## 1. 디자인 철학

> **화면의 95% = 영상.** 메뉴는 학습자에게 방해입니다.

- **다크 모드 강제** (`#0A0A0A`) — 영상 몰입
- 컨트롤 UI는 마우스 정지 3초 후 자동 페이드아웃
- 폰트 Pretendard, 크기 살짝 크게 (16px → 17px)
- 모션은 부드럽게 (300ms ease-out)
- YouTube보다는 **Loom / Mux Player** 스타일 참조

## 3. /v/[강의ID] — 학생 진입 페이지 (회원가입 전) ⭐

### 3.1 화면 구성

```
ClassAuto

○○○ 교수님이 보낸 강의입니다

📚 디지털 위안화의 이해
    경기대학교 · 현대중국사회의이해
    5분 12초

이 강의를 시청하려면:
✓ 학교 이메일 인증 (또는)
✓ 학습 코드 입력: ABCD-1234

[ 학교 이메일로 시작 ]   ← 큰 골드 채움
[ 학습 코드로 시작 ]     ← outline

처음 사용하시나요? [1분 안내 보기]

────────────────
ClassAuto는 학생 학습 도구입니다.
학생 데이터 보호 정책 보기 →
```

### 3.2 핵심 요소
- "○○○ 교수님이 보낸 강의입니다" — 교수자 신원으로 신뢰
- 강의 메타 (제목·소속·길이) 명확히
- 진입 방식 두 가지 (이메일 / 학습 코드)
- 처음 사용자를 위한 "1분 안내"
- 데이터 보호 안내 링크

### 3.3 인증 옵션 분기
- **학교 이메일**: `.ac.kr`, `.edu` 등 도메인 화이트리스트 매칭. 매칭 실패 시 "기관 인증이 필요한 강의입니다" 안내
- **학습 코드**: 교수자가 발급한 4-4 코드. 임시 익명 접속 (시청 데이터는 수집되지만 학생 신원은 비공개)

## 4. 학생 회원가입 흐름

### 4.1 단계
1. **학교 이메일 입력** → 인증 메일 발송
2. **인증 링크 클릭** → 추가 정보 입력 화면
3. **학번·이름·학교·학과 입력** (학과는 자동 매칭 시도)
4. **소셜 로그인 옵션**: 카카오·구글 (Phase 3+)
5. **(선택) 미성년 보호자 동의** — 만 14세 미만 시 필수

### 4.2 디자인 원칙
- 한 화면 한 액션 (인지 부하 최소화)
- 프로그레스 바 (1/3, 2/3, 3/3)
- 학생 부담 적게 — 5개 필드 이내
- 학교·학과 자동완성 (한국 대학 DB 기반)

### 4.3 데이터 정책 사전 고지
가입 시 명시적으로 노출:
- 어떤 데이터를 수집하는가 (영상 시청, Q&A, 출석)
- 누가 볼 수 있는가 (해당 강의 교수자만)
- 광고 미사용
- 졸업 후 자동 삭제

## 6. 영상 시청 화면 (메인)

### 6.1 기본 상태

```
┌────────────────────────────────────────────┐
│ ◀ 현대중국사회의이해 · 3주차          ⚙️  │  ← 미니멀 상단바 (페이드아웃)
├────────────────────────────────────────────┤
│                                            │
│         [AI 아바타 강의 영상]               │
│         (전체 화면 차지)                    │
│                                            │
│                              ┌────────┐    │
│                              │💬 질문  │    │  ← 우측 하단 플로팅 FAB
│                              └────────┘    │
├────────────────────────────────────────────┤
│ ▶ ━━━●━━━━━━━━━━━━━━━━━ 12:34 / 28:00 ⛶ │  ← 미니 컨트롤
│ 👍🤔😊💡  익명 반응 (4개 이모지)            │
└────────────────────────────────────────────┘
```

### 6.2 익명 반응 (4 이모지)
- 👍 좋아요 / 🤔 어렵네요 / 😊 재미있어요 / 💡 알겠어요
- 클릭 시 영상 시점에 태그되어 교수자 "재생 구간 히트맵"에 누적
- 익명 — 교수자도 "누가 눌렀는지"는 모름

## 7. Q&A 사이드 패널

### 7.1 패널 등장 (질문 버튼 클릭 시)

영상 자동 일시정지 → 우측에서 400px 패널 슬라이드인 → 영상은 좌측 60% 폭으로 축소.

```
┌─────────────────────────┬───────────────────┐
│                         │ 💬 강의 도우미     │
│   [영상 - 일시정지]     │ ─────────────────│
│   현재 슬라이드 7        │                   │
│                         │ 🤖 안녕하세요!    │
│                         │    무엇이 궁금    │
│                         │    하신가요?      │
│                         │           👤 GDP와│
│                         │           GNP 차이│
│                         │           가 뭐죠?│
│                         │                   │
│                         │ 🤖 슬라이드 7페이지│
│                         │    내용 기준으로   │
│                         │    답변드리면...   │
│                         │    [출처: p.12-13]│
│                         │                   │
│                         │ ─────────────────│
│                         │ [🎤] [질문 입력]  │
│                         │                   │
│                         │ ⚠ 강의 범위 외    │
│                         │   교수님께 전달   │
└─────────────────────────┴───────────────────┘
```

### 7.2 핵심 인터랙션

**(1) AI 답변의 컨텍스트 인용**
- 모든 답변에 "[출처: 슬라이드 12-13]"
- 클릭 시 영상이 해당 시점으로 점프
- RAG 기반임을 시각적으로 인지

**(2) 음성 질문 (모바일 친화)**
- 마이크 길게 누름 → STT 변환 → 텍스트로 질문
- 외국인 유학생의 한국어 타이핑 부담 해결

**(3) 범위 외 질문 처리** ([02-guardrails.md](./02-guardrails.md) 2차 가드레일)
- AI가 "이 질문은 강의 범위 밖이에요" 판단 시
- [교수님께 전달] / [그래도 답변 받기 (제한적)]
- 교수님께 전달 시 → 교수자 Q&A 인박스로 알림

**(4) Q&A 한도 표시**
```
편당 Q&A: 23 / 100  ·  오늘 남은 질문: 12 / 30
```

## 8. 집중 경고 3단계 + 인터스티셜 퀴즈

### 8.1 1단계 경고 (5초 무반응)

```
░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  ← 화면 가장자리만 부드러운 펄스
░          [영상 계속 재생 중]      ░     (회갈색 글로우)
░  ┌───────────────────────────┐    ░
░  │ 🦉 잠깐, 잘 따라오고       │    ░  ← 토스트 (좌하단)
░  │    계신가요?               │    ░     자동 사라짐 5초
░  │    화면을 클릭해주세요      │    ░
░  └───────────────────────────┘    ░
░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░
```

- 영상은 계속 재생 (방해 최소화)
- 올빼미 표정: 걱정스러운 눈
- 화면 클릭 / 키보드 / 마우스 시 즉시 해제

### 8.2 2단계 경고 (15초 무반응)

```
[영상이 반투명 오버레이로 흐려짐]

      ┌──────────────────────┐
      │       🦉              │   ← 큰 올빼미 (200px)
      │      (놀란 표정)       │
      │                      │
      │ 오늘은 좀 피곤하신가요? │
      │ 잠시 쉬어가도 좋아요   │
      │                      │
      │ [계속 볼게요]         │   ← 클릭 필수
      └──────────────────────┘
```

- 영상 자동 일시정지
- 올빼미: 놀란 큰 눈 (좌우 미세 흔들림 애니메이션)
- 반드시 클릭해야 영상 재개

### 8.3 3단계 경고 (30초 또는 누적 3회)

```
[영상 완전 일시정지 + 다크 오버레이]

     ┌────────────────────────────┐
     │         🦉                  │
     │       (또렷한 응시)         │
     │                            │
     │  집중이 어려우신 것 같아요  │
     │  어떻게 해드릴까요?         │
     │                            │
     │  [처음부터 다시 보기]      │
     │  [짧은 퀴즈로 점검하기]    │
     │  [5분 후 다시 시도]        │
     └────────────────────────────┘
```

- 영상 완전 정지
- 올빼미: 또렷한 응시 (눈동자 정중앙 고정)
- 3가지 액션 명시적 선택 강제
- 누적 기록 → 교수자 대시보드 "주의 필요 학습자"에 자동 등록

### 8.4 인터스티셜 퀴즈 (영상 중간 자동)

```
┌──────────────────────────────────────┐
│ 🦉 잠깐 확인! (10초)                  │
│ ─────────────────────────────────── │
│                                      │
│ 방금 들으신 내용 중,                  │
│ 디지털 위안화의 가장 큰 특징은?        │
│                                      │
│ ┌─ A. 익명성 보장 ────────────┐       │
│ │  B. 추적 가능한 중앙은행 발행 │  ✓   │
│ │  C. 블록체인 분산 원장        │       │
│ │  D. 비트코인과 동일한 구조    │       │
│ └─────────────────────────────┘       │
│                                      │
│            [정답 확인]                │
└──────────────────────────────────────┘
```

- **정답 시**: ✓ 녹색 + "잘 이해하고 계세요!" + 2초 후 영상 자동 재개
- **오답 시**: "정답은 B예요. 이 부분은 슬라이드 8-9에서 다뤘어요" + [해당 부분 다시 보기] / [그냥 계속 볼게요]
- 다시 보기 클릭 시 해당 시점으로 점프 + 재시청
- 오답 기록 → 교수자 "취약점 히트맵"에 누적

### 8.5 디자인 디테일
- 모든 오버레이는 카드 형태 (radius 16px, 큰 여백)
- 올빼미 일러스트는 단색 SVG (회갈색)
- 모션은 부드럽게 (개입이라기보다 환기)
- 1→2→3단계로 갈수록 화면 점유율·색채 대비 증가

## 9. /profile (학습자 마이페이지)

### 9.1 화면 구성

```
👤 ○○○ 학생
   경기대학교 · 중어중문학과 · 3학년

   [잔디 그리드 - GitHub 스타일]

📚 수강 중 (3)
   - 현대중국사회의이해 (78% 진행)
   - 기초중국어듣기 (45%)
   - 글로벌문화의이해 (12%)

   각 강의별 인증서 PDF 다운로드 가능

📊 누적 통계
   총 학습 시간: 23시간
   평균 정답률: 82%

   받은 격려: 12회

⚙️ 설정
   언어 / 알림 / 접근성 / 계정 관리
```

### 9.2 학습자 동기부여 핵심 요소
- **스트릭 시각화** (Duolingo 잔디) — 매일 방문 욕구 자극
- **수강 완료 인증서** PDF — 졸업·취업용 활용 가능
- **격려 메시지** — 교수자가 보낸 자동 격려가 누적

## 10. 다국어 인터페이스 (Phase 2~3)

### 10.1 지원 언어
- 한국어 (기본)
- 중국어 간체/번체 (어흥 교수님 SEP 자매대학 학생들)
- 영어 (글로벌 + 일반 외국인 유학생)

### 10.2 언어 선택 위치
- 학생 가입 시 1회 선택
- /profile 설정에서 언제든 변경
- 우측 상단 언어 토글 (작게)

### 10.3 Q&A 답변 언어 선택
- 영상은 한국어, 답변은 중국어 가능
- 학생 설정에서 "답변 언어" 별도 선택
- 자동 번역 시 "원문(한국어) 보기" 옵션

### 10.4 어흥 교수님 학과 직접 효과
SEP 100+ 자매대학 학생들이 한국어 학습 중. 다국어 인터페이스 갖추면 즉시 사용 가능한 도구.

## 11. 접근성 (a11y)

### 11.1 영상 시청 화면
- 자동 자막 (다국어 자막 기능 활용 + 한국어 자막 필수)
- 키보드 단축키 전체 지원
  - Space: 재생/일시정지
  - J / L: 10초 뒤/앞
  - K: 일시정지
  - M: 음소거
  - F: 전체 화면
  - C: 자막 토글
- 화면 낭독기(NVDA, JAWS) 호환

### 11.2 Q&A 챗봇
- 음성 입력 (시각장애 학생에게 결정적)
- 음성 출력 (TTS 활용)
- 큰 글씨 옵션
- 색맹 친화 색상 (골드 단색 + 청록 보조)

### 11.3 페이지 자체
- WCAG 2.1 AA 준수
- 접근성 설정 메뉴 (글씨 크기·대비·모션 줄이기)
- `prefers-reduced-motion` 지원 (동적 요소 다 끄기)

### 11.4 추가 가치
한국 대학 일부는 장애학생 지원 도구를 우선 도입 → 도입 우선순위 상승.
