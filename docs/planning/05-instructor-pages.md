## 3. 교수자 첫 사용 온보딩

### 3.1 환영 모달 (첫 로그인 시)
- "안녕하세요, ○○○ 교수님"
- 5단계 가이드 미리보기
- "지금 시작" / "둘러보기"

### 3.2 빈 상태 디자인 (Empty State)
- 통계 카드는 모두 "—" (의미 있는 placeholder)
- 메인 영역에 큰 "첫 강의 영상 만들기" CTA
- 샘플 데이터로 미리보기 가능한 토글 ("샘플 강의로 둘러보기")

### 3.3 첫 영상 제작 가이드 체크리스트
사이드 패널에:
- [ ] PPT 업로드
- [ ] 스크립트 검토
- [ ] AI 아바타 선택
- [ ] 영상 생성
- [ ] 학생에게 공유

각 단계 완료 시 ✓로 변경, 100% 완료 시 축하 모달.

### 3.4 학과/강의 정보 입력
- 소속 학교, 학과
- 가르치는 과목명
- 학기 시작일
- 학생 수 (선택적 — 정확한 한도 안내용)

이 데이터가 있어야 대시보드의 "이번 학기 ○주차" 같은 컨텍스트가 작동.

## 4. /dashboard (홈)

### 4.1 상단 영역
- **컨텍스트 바**: "안녕하세요, 하두진 교수님 · 2026 봄학기 · 현대중국사회의이해" + 학기 전환 셀렉트 + 검색
- **인사 카드**: "이번 주 새 질문 12개, 시청 미완료 학습자 7명" 한 줄 요약

### 4.2 메트릭 카드 6개 (2행 × 3열)

| 카드 | 표시 내용 | 색상 의미 |
|---|---|---|
| 시청 완료율 | 78% (전주 대비 ▲4%p) | 녹색 화살표 |
| 평균 정답률 | 82% | 무채색 |
| 미응답 Q&A | **5건** (클릭 시 인박스 점프) | 빨강(액션) |
| 활성 학습자 | 47/52명 | 무채색 |
| 이번 달 영상 | 8 / 20편 (Pro) | 진행 바 |
| 누적 사용 비용 | $46.20 (한도 $200) | 진행 바 |

### 4.3 메인 차트 (좌측 2/3)
- **강의별 시청 추이** — 라인 차트(주차별), 강의 3~5개 토글
- ElevenAgents Success Rate 차트 톤 (얇은 라인, 무채색 + 그린 한 포인트)

### 4.4 우측 "주의 필요" 위젯 (1/3)
- **답변 대기 질문** (최신 5건) → 클릭 시 Q&A 인박스
- **시청 부진 학습자** (3일 이상 미시청, 5명) → 알림 발송
- **자주 멈춘 구간** (가장 많이 일시정지/되감기된 슬라이드 Top 3) → 다음 강의 보강 추천

### 4.5 하단
**최근 강의 영상 그리드** (썸네일 카드 4개) — 썸네일 + 강의명 + 시청 완료율 + Q&A 수 + 메뉴.

### 4.6 동적 요소
- 통계 카드 카운트업 애니메이션 (페이지 진입 시 1회)
- 변화율 옆 sparkline (지난 7일 추이)
- 카드 hover 시 sparkline 활성화

## 5. /studio (영상 제작 마법사) ⭐ 핵심

### 5.1 진입 흐름
"새 영상 만들기" 버튼 → **PPT 업로드 모달** → 자동 분석 시작 → **3단 마법사 화면**

### 5.2 3단 레이아웃

```
┌──────────────┬──────────────────────────┬──────────────────┐
│ 슬라이드 패널 │ 중앙 작업 영역            │ 우측 설정 패널    │
│ (240px)      │ (가변)                   │ (320px)          │
├──────────────┼──────────────────────────┼──────────────────┤
│ Slide 1 ●    │ [현재 슬라이드 미리보기]  │ ▼ 아바타          │
│ Slide 2      │                          │   김교수 페르소나 │
│ Slide 3 ⚠   │ ┌──────────────────────┐ │                  │
│ Slide 4      │ │ 슬라이드 이미지       │ │ ▼ 음성 (이중 TTS)│
│              │ └──────────────────────┘ │   주: 한국어 (여) │
│              │                          │   부: 中文 (남)   │
│              │ ┌── 스크립트 에디터 ──┐ │                  │
│              │ │ 원본 PPT 노트:       │ │ ▼ 강의 설정       │
│              │ │ "GDP는 국내총생산.." │ │   링크 만료: 30일 │
│              │ │ ──────────────────── │ │   집중 경고: 켜짐 │
│              │ │ AI 다듬은 스크립트:   │ │   퀴즈 난이도: 중 │
│              │ │ "안녕하세요. 오늘은  │ │                  │
│              │ │  국내총생산..."       │ │ ▼ Q&A 범위       │
│              │ │ [채택] [거부] [편집] │ │   업로드 자료만   │
│              │ └──────────────────────┘ │                  │
│              │                          │ ─────────────── │
│              │                          │ 💰 예상 비용     │
│              │                          │   TTS: $2.10    │
│              │                          │   아바타: $5.10  │
│              │                          │   Total: $7.20  │
└──────────────┴──────────────────────────┴──────────────────┘
       [← 이전 슬라이드] [다음 →] [전체 생성 시작]
```

### 5.3 핵심 인터랙션 4가지

**(1) 인라인 diff 스크립트 편집**
- PPT 발표자 노트(원본) vs AI 다듬은 스크립트(제안) 위아래 표시
- 슬라이드별로 [채택] / [거부] / [수동 편집]
- 채택 시 슬라이드 패널의 해당 슬라이드 ● 점등
- ⚠ 표시: AI가 "정보 부족, 보강 필요" 판단한 슬라이드

**(2) 실시간 비용 미터**
- 우측 하단 항상 노출
- 슬라이드 추가/삭제 → 즉시 비용 재계산 애니메이션
- 클릭 시 분석 펼쳐짐 (TTS 글자수, 아바타 초수, API 단가)
- 한도 80% 초과 시 빨강 경고

**(3) 미리듣기 (Preview)**
- 각 슬라이드 옆 ▶ 버튼 → 해당 문장만 TTS 미리듣기
- 톤 조절 슬라이더 (속도/높이)

**(4) 생성 진행 모달**
4단계 프로그레스:
1. 스크립트 검토 완료 ✓
2. TTS 음성 생성 중... (12/24 슬라이드, 예상 3분)
3. AI 아바타 영상 합성 대기
4. 최종 인코딩 대기

백그라운드 실행 가능 ("완료 시 이메일 알림" 토글).

### 5.4 영상 완성 후 공유 패널 ⭐

```
✨ 강의 영상이 완성되었어요!
강의명: 디지털 위안화의 이해 (5분 12초)
강의 링크: classauto.live/v/abc123XYZ

학생들에게 공유하세요
[📧 이메일] [💬 카톡] [🐦 X] [🔗 URL] [📱 QR]

📊 학습 코드: ABCD-1234
(학생이 코드 입력으로도 접근 가능)

⚙️ 공유 설정
[ ] 비밀번호 보호
[✓] 학교 이메일만 시청 가능
[ ] 시청 기간 제한 (학기말까지)
```

### 5.5 QR 코드 핵심 역할
- 강의실 PPT 마지막 페이지에 QR → 학생 스캔으로 즉시 접속
- 인쇄물(시험·과제 안내)에 QR 삽입
- 학교 게시판·복도 포스터에 부착
- 모바일 우선 학습 환경에 최적

QR 디자인:
- 정사각형 PNG 1024x1024
- 중앙에 ClassAuto 로고
- 골드 그라데이션 옵션 (강의실 슬라이드용)
- 학생 화면이 QR 스캔 → 자동 로그인 또는 익명 접속

### 5.6 디자인 특징
- 좌측 슬라이드 패널 좁고 빽빽 (Adobe Premiere 스타일)
- 중앙은 여백 충분 (작업 집중)
- 우측 패널은 아코디언 (필요시 펼침)
- 다크/라이트 토글 (긴 작업이라 다크 옵션 필수)

## 6. /inbox (Q&A 인박스)

Gmail 스타일 3단:

### 6.1 좌측: 강의별 필터
- 강의 목록 + 안 읽음 카운트
- "전체" 필터

### 6.2 중앙: 질문 리스트
3개 탭:
- AI 자동응답 (참고용 모니터링)
- 교수자 응답 필요 (액션 아이템)
- 범위 외 거부 (학생이 "교수님께 전달"한 것)

각 항목: 학생명·강의명·시간 + 질문 미리보기.

### 6.3 우측: 선택한 질문의 컨텍스트
- 어느 영상 어느 시점에서 질문했는가
- 학생이 본 슬라이드 미리보기
- AI 답변 초안
- **유사 질문 클러스터링 (Pro)**: "이 질문은 다른 학생 12명도 비슷하게 물었습니다 → 다음 강의 보강 추천"

## 7. /analytics (분석 리포트)

### 7.1 재생 구간 히트맵 (Pro)
- 영상 타임라인 위에 색 농도 (어느 구간을 반복 재생/건너뛰기)
- 영상별로 분리, 강의 단위 합산도 가능

### 7.2 취약점 히트맵 (Pro)
- 강의 챕터 × 학습자 매트릭스
- 정답률 낮은 셀이 빨갛게
- 행렬 정렬: 학습자별 / 챕터별 / 정답률 순
- "이 챕터를 보강하시겠습니까?" 액션

### 7.3 정답률·Q&A·비용 미터 (Basic+)
- 강의별 정답률 추이
- Q&A 트렌드 (자주 묻는 토픽)
- 월별 비용 사용 추이

### 7.4 익명 반응 집계 (Basic+)
- 영상별 4가지 이모지 반응 비율
- 시점별 반응 분포

### 7.5 딴짓 포착 리포트 (Basic+)
- 의심 패턴 학습자 목록
- 시청 시간 < 영상 길이 30% 학습자
- 매크로 의심 행동 (가드레일 4차 데이터)

### 7.6 대면 수업 브리핑 자동 (Pro)
- 영상 시청 데이터 기반 자동 생성
- "지난 영상에서 학생들이 가장 어려워한 부분: ○○○"
- "다음 대면 수업에서 다룰 주제 추천: △△△"
