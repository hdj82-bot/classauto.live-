## 1. 핵심 원칙

### 1.1 가격 설계 철학
- **편수 + 학생 수 이중 한도**로 사용량 통제
- 학계 보급 우선 — 가격은 낮게, 기능은 높게
- 영상 1편 생성 원가 약 ₩11,000 (HeyGen + ElevenLabs + Claude API + DeepL)
- Basic 플랜 (₩19,000 / 8편) = 편당 ₩2,375 — 운영비 회수 + 학계 보급 우선

### 1.2 단순성 원칙
- 크레딧 시스템 없음 (이전 충전팩 모델 폐기)
- 분 단위 과금 없음 (편수만 카운트)
- 학생 수 카운트는 MAU(Monthly Active Learners) 기준

### 1.3 정직성 원칙 — "한도 투명성" (2026-05-06 갱신)

이전 표현: "비용 투명성"
수정 표현: **"한도 투명성"**

이유: ClassAuto는 구독제(월 정액)이므로 교수자에게 비용($/₩/토큰)을 노출하지 않습니다.
대신 **편수·학생 수 한도**를 매 화면에서 명확하게 보여줍니다.

✅ 표시할 것 (한도 단위):
- "이번 달 영상: 8 / 20편 (Pro)"
- "활성 학습자: 47 / 150명"
- "이번 강의 Q&A: 23 / 100"
- "사용 가능 슬라이드: 248 / 500개"

❌ 표시 안 할 것 (비용 단위):
- "이번 달 사용: $46.20 / $200.00"
- "TTS 비용: $2.10"
- "AI 아바타: $5.10"
- "Total: $7.20"

**왜 이렇게 바꾸나**:
- 구독제 본질 = "마음껏 써도 됨"
- 비용 표시는 자유로운 사용 저해
- "내가 얼마 쓰는지" 신경 쓸 이유 없음
- 한도만 알면 충분

원가 모니터링은 ClassAuto 운영진의 admin 페이지에서만 수행.

### 1.4 한도 단위로 통합
- 모든 한도를 카드와 비교표에 명시
- 해지·다운그레이드·환불 정책을 페이지에 노출
- 기관 라이선스만 별도 협의

## 4. 결제·해지 정책

### 4.1 월/연 결제
- 페이지 토글 기본값 = **연 결제** (Anchoring 효과로 전환율 상승)
- 토글 변경 시 가격 숫자가 0.3초 ease-out으로 부드럽게 전환

### 4.2 결제 수단
- 신용카드 (Visa, MasterCard, AMEX)
- 카카오페이, 토스페이, 네이버페이
- 학교/기관 결제: 세금계산서 발행 + 계좌 이체

### 4.3 해지
- 클릭 1번으로 해지 가능 (Stripe Customer Portal 연동)
- 해지 후 결제 주기 끝까지 사용 가능
- 해지 후 데이터 보존 기간: **60일**
- 재가입 시 60일 이내면 데이터 자동 복구

### 4.4 다운그레이드

**Pro → Basic**
- 기존 영상 그대로 유지 (워터마크 추가 없음)
- Pro 전용 기능(취약점 히트맵 등)으로 만든 데이터는 읽기 전용으로 보존
- 학생 수 80명 초과 시 → 다음 결제 주기까지 유예 (해당 기간 신규 등록 불가)
- 다음 결제 주기부터 Basic 요금 적용

**Basic → Free**
- 기존 영상에 워터마크 자동 추가
- 8편 초과 영상은 비공개 처리 (삭제 아님)
- Free 한도 내로 정리 시 다시 공개 가능

### 4.5 환불
- 결제 후 7일 이내, 해당 기간 동안 영상을 생성하지 않은 경우 100% 환불
- 7일 이후 또는 영상 생성 시작 후: 환불 불가
- 문의: hello@classauto.live

### 4.6 데이터 Export 권리
- **모든 플랜**: 강의 영상 mp4 다운로드 (해지 후 60일까지)
- **Pro 전용**: 학생 데이터 CSV (개인정보 제거 옵션)
- **Pro 전용**: Q&A 로그 (논문 연구용 익명화 데이터)
- 해지 후 30일까지 export 권리 유지

## 6. FAQ (요금 페이지 게재용)

1. **월 8편/20편을 다 못 쓰면 다음 달로 넘어가나요?**
   이월되지 않습니다. 한국 대학 교수자 평균 3~4과목 기준 한 달 8편이면 충분하도록 설계했습니다.

2. **Free에서 Basic으로 업그레이드하면 그동안 만든 영상은?**
   기존 영상은 그대로 유지되며, 워터마크가 자동으로 제거됩니다. 영상 링크도 변하지 않습니다.

3. **학생 수 제한이 있나요?**
   네. 월간 활성 학습자(해당 월에 영상을 시청한 학생) 기준으로 Free 30명, Basic 80명, Pro 150명까지 지원합니다. 더 큰 강의는 기관 라이선스로 무제한 이용 가능합니다.

4. **플랜을 중간에 변경할 수 있나요?**
   언제든지 가능합니다. 업그레이드는 즉시 반영, 다운그레이드는 다음 결제 주기부터 적용됩니다.

5. **베타 기간엔 어떤 혜택이 있나요?**
   Phase 2 베타 기간(2026) 학계 대학 교수자 누구나 Pro 기능 전체를 무료로 이용하실 수 있습니다. 베타 참여자는 정식 출시 후 평생 30% 할인이 적용됩니다.

6. **결제 수단은 어떤 것을 지원하나요?**
   신용카드(Visa, MasterCard, AMEX), 카카오페이, 토스페이, 네이버페이를 지원합니다. 학교/기관 결제의 경우 세금계산서 발행 및 계좌 이체도 가능합니다.

7. **환불 정책은 어떻게 되나요?**
   결제 후 7일 이내, 해당 기간 동안 영상을 생성하지 않은 경우 100% 환불 가능합니다. 자세한 사항은 hello@classauto.live로 문의해주세요.

8. **Q&A 챗봇으로 학습 외 질문도 할 수 있나요?**
   아니요. ClassAuto의 Q&A는 강의 자료 기반 RAG 방식으로 동작하며, 강의 범위 밖 질문은 자동으로 거부됩니다(유사도 임계값 0.7). 자세한 정책은 [02-guardrails.md](./02-guardrails.md) 참조.

9. **학생이 영상을 틀어놓고 자리를 비우면 어떻게 되나요?**
   영상 시청 중 5~15분 간격으로 인터스티셜 퀴즈가 등장합니다. 30초 내 응답하지 않으면 영상이 자동 일시정지되고, 누적 3회 미응답 시 해당 시청은 무효 처리됩니다. 동일 학생의 동시 재생도 1개로 제한됩니다.

10. **영상 1편을 만들 때 얼마가 청구되나요?** (2026-05-06 추가)
    구독제이므로 영상 1편당 별도 비용은 없습니다. 월 정액(Basic ₩19,000 / Pro ₩45,000) 안에서 편수 한도(8편 / 20편)까지 자유롭게 만드실 수 있습니다.
