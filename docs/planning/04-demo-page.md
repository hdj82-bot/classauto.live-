## 1. 페이지의 단 하나의 목적

> "이 페이지에 들어온 잠재 교수자가, **3분 안에**, AI 강의 영상이 학생과 어떻게 양방향으로 소통하는지 **관찰자 입장에서 확인**하고, **'아, 이건 진짜 다르네'** 라고 느끼게 만들 수 있는가?"

3분 안에 그 감각이 안 오면 페이지는 실패. 모든 디자인·콘텐츠·인터랙션 결정의 기준.

### 핵심 메시지 (2026-05-06 갱신)
- ❌ 잘못된 메시지: "교수님, 학생이 되어보세요" (교수자에게 학생 역할 강요)
- ✅ 올바른 메시지: "ClassAuto는 강의 영상이 학생에게 답합니다" (양방향 학습 본질 강조)
- 페르소나: 교수자가 **관찰자**로서 학생 측 화면을 미리 확인

### 2차 목적
- 회원가입 전환 (베타 신청)
- 동료 교수자에게 공유 (입소문)
- 학과장에게 보여주기 (기관 도입)

## 5. 분야 선택 (두 버전 데모) — 2026-05-06 갱신

### 5.1 두 카드 UI

```
인문계열                          자연계열
A · LIBERAL ARTS                 B · NATURAL SCIENCE
中 (한자 또는 책 SVG)             🌱 (잎사귀 SVG)

중국어문법의 이해                  광합성의 원리
[부제 5.3 참조]                   [부제 5.3 참조]
8 슬라이드 · 2 추천 질문 · 1 퀴즈  8 슬라이드 · 2 추천 질문 · 1 퀴즈
[시작하기 →]                     [시작하기 →]
```

### 5.2 주제 선택 근거

| 주제 | 선택 이유 |
|---|---|
| **중국어문법의 이해 (把자문)** | 어흥 교수님 전공 분야 직결 (진정성), 중국어 입문자에게 친숙, 어순 도식 시각화 우수 |
| **광합성의 원리** | 한국인 모두가 학창 시절 학습한 보편적 주제, 진입 장벽 낮음, 화학식 시연 효과적 |

이전 후보(현대중국사회의이해, 특수상대성이론) 폐기 이유:
- 사회 이슈는 시의성에 따라 데모가 빠르게 낡음
- 상대성이론은 너무 어려워 일반 교수자 진입 장벽

### 5.3 카드 부제

**A 인문계열**:
"把자문(把字句)의 어순 규칙을 중심으로 — 추천 질문, 출처 인용, 그리고 강의 외 질문 거부까지 차례로 시연됩니다."

**B 자연계열**:
"빛 에너지가 화학 에너지로 전환되는 과정을 중심으로 — 추천 질문, 출처 인용, 그리고 강의 외 질문 거부까지 차례로 시연됩니다."

### 5.4 두 영상의 공통 시간선

| 시점 | 시연 요소 | 영상 A: 중국어문법 | 영상 B: 광합성 |
|---|---|---|---|
| 0:00 | 영상 시작 | 把자문 도입 | 광합성 정의·필요성 |
| 1:30 | 추천 질문 1 | "把자문은 언제 사용하나요?" | "광합성과 호흡의 차이는?" |
| 2:00 | 인터스티셜 퀴즈 | 把자문 처치 대상 4지선다 | 광합성 결과물 4지선다 |
| 2:30 | 토스트 안내 | (강의 외 질문 시도 권유) | (동일) |
| 3:30 | 추천 질문 2 | "일반 어순과 어떻게 다른가요?" | "엽록체는 왜 녹색인가요?" |
| 5:00 | 영상 종료 | CTA 모달 | CTA 모달 |

**왜 두 버전인가**: 인문계 교수자에게는 중국어문법이 친숙, 이공계 교수자에게는 광합성이 "내 분야도 되네" 시연. 분야 선택 자체가 페르소나 데이터로 활용.

## 7. AI 답변의 4가지 차별 시각화

응답 도착 시마다 다음 4가지를 시각적으로 보여줌 (예: 인문계 把자문):

```
🤖 把자문은 동작의 처치 대상을 동사 앞으로
   끌어내어 강조하는 구조입니다. 일반 어순
   "주어+동사+목적어"와 달리...

   ┌──────────────────────────────────┐
   │ 📍 출처: 슬라이드 5-6            │ ← 클릭 시 영상 점프
   │ ⏱️ 영상 시점: 1:48-2:24          │
   └──────────────────────────────────┘

   [🔄 다시 듣기]  [📌 이 부분 다시 보기]
```

**4요소**:
1. 출처 슬라이드 명시 (RAG 신뢰성)
2. 영상 시점 점프 가능 (학습 흐름 연결)
3. "다시 듣기" 버튼 (TTS 라이브 시연)
4. "이 부분 다시 보기" (영상 점프)

### 9.1 인문계열 (中国语文法 把자문)

```

○ A. 동작의 주체
● B. 동작의 대상 (처치 대상)  ← 정답
○ C. 시간 표현
○ D. 장소 표현

```

### 9.2 자연계열 (광합성)

```
🦉 잠깐 확인! (10초)

광합성의 결과물로 옳은 것은?

○ A. 이산화탄소
● B. 포도당과 산소  ← 정답
○ C. 질소
○ D. 메탄

[정답 확인]
```

### 9.3 정답·오답 처리
- **정답 시**: ✓ "잘 이해하고 계세요!" → 2초 후 영상 자동 재개
- **오답 시**: "정답은 B예요. 슬라이드 X-X를 다시 볼까요?" + [해당 부분 다시 보기] / [그냥 계속 볼게요]

영상이 그냥 영상이 아니라는 걸 깨닫는 결정적 모먼트.

## 12. 인증 없는 세션 관리

```
페이지 진입 시:
1. 브라우저 fingerprint + IP → 임시 세션 ID 생성
2. Redis에 24시간 TTL로 세션 저장
3. 쿠키에 세션 ID 저장 (httpOnly, sameSite=strict)

세션 데이터:
- 사용한 질문 수 (0~3)
- 시작 시각
- 사용자 입력 로그 (개선 분석용, 익명)
```

봇·남용 방지: Cloudflare Turnstile, IP당 1일 3 세션, 같은 질문 반복 차단, 일일 비용 캡 ($100/일).

데이터 보호: 24시간 후 자동 삭제, 익명화 후 모델 개선용으로만, IP·fingerprint는 봇 차단 외 사용 금지.


## 18. FAQ (데모 한정 4개)

1. **회원가입 없이 정말 다 써볼 수 있나요?** — 네, 영상 시청과 AI 챗봇과의 대화까지 회원가입 없이 가능합니다.
2. **내 강의로도 이런 영상이 만들어지나요?** — 네, PPT 한 장만 있으면 됩니다.
3. **학생 입장 화면이 실제와 같나요?** — 본 페이지에서 보시는 화면은 실제 학생이 보는 화면과 동일합니다.
4. **두 데모 영상 모두 확인할 수 있나요?** — 한 세션당 한 영상이지만, "한 번 더 확인" 옵션으로 다른 영상도 시도 가능합니다.
