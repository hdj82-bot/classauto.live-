## 1. 페이지의 단 하나의 목적

> "이 페이지에 들어온 잠재 교수자가, **3분 안에**, 학생 입장이 되어보고, **'아, 이건 진짜 다르네'** 라고 느끼게 만들 수 있는가?"

3분 안에 그 감각이 안 오면 페이지는 실패. 모든 디자인·콘텐츠·인터랙션 결정의 기준.

### 2차 목적
- 회원가입 전환 (베타 신청)
- 동료 교수자에게 공유 (입소문)
- 학과장에게 보여주기 (기관 도입)

## 2. 핵심 사용자 여정

```
[0:00] 페이지 진입 → 무엇을 해야 하는지 즉시 이해
[0:05] 분야 선택 (사회과학 / 자연과학)
[0:10] 영상 자동 재생 시작 (다크 모드)
[1:00] 토스트 등장: "강의 외 질문 시도해볼까요?" (8초 후 사라짐)
[1:30] 추천 질문 클릭 → AI 답변 + 출처 인용 시연 ⭐
[2:00] 인터스티셜 퀴즈 등장 (10초 카운트다운)
[2:30] 자유 질문 입력 → 음성 재생 옵션
[3:00] "마지막 질문이에요" 안내
[3:30] 강의 외 질문 시도 → 자동 거부 시연 ⭐⭐
[5:00] 영상 종료 → CTA 모달 풀스크린
[CTA 닫기 후] 타임랩스 1분 영상 자동 노출
```

매 30초마다 새로운 차별점이 노출되도록 설계.

## 4. 미니 히어로

### 4.1 카피
- 헤드라인 (Paperlogy ExtraBold): **"교수님, 30초만 학생이 되어보세요."**
- 서브 (Pretendard): "실제 강의 영상을 보면서 AI 튜터에게 질문해보세요. 회원가입 없이 바로 시작하실 수 있습니다."
- 메타: "⏱️ 약 3분 소요 · 💬 질문 3건 무료 · 📱 모바일 가능"
- CTA: **"▶ 체험 시작하기"** (단일 골드 채움 버튼)

### 4.2 진입 즉시 효과
페이지 로드 시 미리보기 영상이 흐릿하게 자동 재생되다 5초 후 일시정지. "이미 시작된 듯한" 심리 효과.

## 5. 분야 선택 (두 버전 데모)

### 5.1 두 카드 UI

```
사회과학                     자연과학·공학
🌏                          📐
현대중국사회의이해 (5분)        특수상대성이론 입문 (5분)
[▶ 시작하기]                 [▶ 시작하기]
```

### 5.2 두 영상의 공통 시간선

| 시점 | 시연 요소 | 영상 A: 중국사회 | 영상 B: 상대성이론 |
|---|---|---|---|
| 0:00 | 영상 시작 | 디지털 위안화 도입 | 광속 불변 원리 도입 |
| 1:30 | 추천 질문 1 | "GDP·GNP 차이?" | "광속이 왜 일정한가요?" |
| 2:00 | 인터스티셜 퀴즈 | 디지털 위안화 4지선다 | 시간 팽창 4지선다 |
| 2:30 | 토스트 안내 | (강의 외 질문 시도 권유) | (동일) |
| 3:30 | 추천 질문 2 | "미중 무역분쟁 영향?" | "쌍둥이 역설이란?" |
| 5:00 | 영상 종료 | CTA 모달 | CTA 모달 |

**왜 두 버전인가**: 사회과학 교수자에게는 중국사회가 친숙, 이공계 교수자에게는 상대성이론이 "내 분야도 되네" 시연. 분야 선택 자체가 페르소나 데이터로 활용.

## 7. AI 답변의 4가지 차별 시각화

응답 도착 시마다 다음 4가지를 시각적으로 보여줌:

```
🤖 GDP는 한 나라 안에서 생산된 모든 재화와
   서비스의 가치를 합한 것이고, GNP는...

   ┌──────────────────────────────────┐
   │ 📍 출처: 슬라이드 7-8            │ ← 클릭 시 영상 점프
   │ ⏱️ 영상 시점: 2:14-3:02         │
   └──────────────────────────────────┘

   [🔄 다시 듣기]  [📌 이 부분 다시 보기]
```

**4요소**:
1. 출처 슬라이드 명시 (RAG 신뢰성)
2. 영상 시점 점프 가능 (학습 흐름 연결)
3. "다시 듣기" 버튼 (TTS 라이브 시연)
4. "이 부분 다시 보기" (영상 점프)

## 8. 강의 외 질문 시연 (결정적 차별점)

### 8.1 토스트 타이밍
**1분 시점**에 토스트 등장 (페이지 진입 직후 X, 적응 후 호기심 최고조 시점):

```
💡 시도해볼까요?
   "오늘 점심 뭐 먹지?" 같은 강의와 무관한 질문도
   입력해보세요. AI가 어떻게 반응하는지 확인할 수 있어요.

   [지금 시도]   [×]
```

8초 후 자동 사라짐 → 우상단에 작은 트로피 배지로 축소:
```
🏆 도전 과제: 강의 외 질문 시도하기
```

클릭 시 토스트 다시 표시. 시도 완료 시 ✓ + "차별점 1개 발견!" 글로우.

### 8.2 자동 거부 답변

```
🤖 이 질문은 강의 범위 밖이에요.
   강의 자료에서 답을 찾을 수 없습니다.

   다른 EdTech AI 챗봇은 이런 질문에도 답하지만,
   ClassAuto는 학생이 학습에 집중하도록 설계되었습니다.
```

답변 자체가 마케팅 메시지가 됨.

## 9. 인터스티셜 퀴즈

영상 2분 지점 자동 등장:

```
🦉 잠깐 확인! (10초)
방금 들으신 내용 중,
디지털 위안화의 가장 큰 특징은?

○ A. 익명성 보장
● B. 추적 가능한 중앙은행 발행
○ C. 블록체인 분산 원장
○ D. 비트코인과 동일한 구조

[정답 확인]
```

- 정답 시: ✓ "잘 이해하고 계세요!" → 2초 후 영상 자동 재개
- 오답 시: "정답은 B예요. 슬라이드 8-9를 다시 볼까요?" + 점프 옵션

영상이 그냥 영상이 아니라는 걸 깨닫는 결정적 모먼트.

### 10.1 횟수 표시
```

```

### 10.2 1건 남았을 때 안내
```
💡 마지막 질문이에요. 강의와 무관한 질문도 한 번 시도해보세요!
   ClassAuto가 어떻게 반응하는지 볼 수 있어요.
```

### 10.3 5건 모두 사용 시

## 12. 인증 없는 세션 관리

```
페이지 진입 시:
1. 브라우저 fingerprint + IP → 임시 세션 ID 생성
2. Redis에 24시간 TTL로 세션 저장
3. 쿠키에 세션 ID 저장 (httpOnly, sameSite=strict)

세션 데이터:
- 사용한 질문 수 (0~3)
- 시작 시각
- 사용자 입력 로그 (개선 분석용, 익명)
```

봇·남용 방지: Cloudflare Turnstile, IP당 1일 3 세션, 같은 질문 반복 차단, 일일 비용 캡 ($100/일).

데이터 보호: 24시간 후 자동 삭제, 익명화 후 모델 개선용으로만, IP·fingerprint는 봇 차단 외 사용 금지.

## 14. 1분 타임랩스 영상

CTA 모달 닫으면 페이지 하단으로 자동 스크롤되며 노출:

```
이 영상은 어떻게 만들어졌을까요?

[60초 타임랩스 - 자동 재생, 음소거]
PPT 업로드 → 스크립트 생성 → AI 아바타 합성 → 완성

⏱️ 실제 소요: 11분 47초
💰 생성 비용: ₩9,500
📝 슬라이드: 8장

[영상 제작 마법사 둘러보기 →]
```

### 14.1 60초 타임랩스 4단계 (각 15초)

| 시점 | 화면 | 메시지 |
|---|---|---|
| 0:00~0:15 | PPT 업로드 → 8장 슬라이드 펼쳐짐 | "PPT 한 장으로 시작합니다" |
| 0:15~0:30 | AI가 스크립트 자동 생성, diff 표시 | "AI가 강의 스크립트를 다듬어드립니다" |
| 0:30~0:45 | TTS + 아바타 합성 진행 바 | "음성과 AI 아바타가 합성됩니다" |
| 0:45~1:00 | 완성 영상 재생 + QR 코드 생성 | "12분 만에 완성. 학생에게 공유하세요" |

음소거 자동 재생: 모바일 자동 재생 가능, 부담 없이 보게 됨, 자막은 큰 텍스트로.

## 15. 공유 영역 (4채널)

```
동료 교수님께도 알려보세요

📧 이메일  💬 카톡  🐦 X  🔗 URL
```

- **이메일**: `mailto:?subject=...&body=...` 미리 작성된 본문
- **카톡**: KakaoLink SDK (썸네일 + 제목 + URL 카드)
- **X**: `https://twitter.com/intent/tweet?text=...&url=...`
- **URL**: 클립보드 복사 + 토스트 "URL이 복사되었어요"

UTM 파라미터 추적: `?utm_source=demo&utm_medium=email|kakao|twitter|url`

## 18. FAQ (데모 한정 4개)

1. **회원가입 없이 정말 다 써볼 수 있나요?** — 네, 영상 시청과 AI 챗봇과의 대화까지 회원가입 없이 가능합니다.
2. **내 강의로도 이런 영상이 만들어지나요?** — 네, PPT 한 장만 있으면 됩니다.
3. **학생 입장 화면이 실제와 같나요?** — 본 데모는 실제 학생이 보는 화면과 동일합니다.
4. **두 데모 영상 모두 체험할 수 있나요?** — 한 세션당 한 영상이지만, "한 번 더 체험" 옵션으로 다른 영상도 시도 가능합니다.
