# Mascot (올빼미 마스코트 가이드 — Legacy)

> **상태**: 확정 · 2026-05-05
> **결정**: 올빼미 캐릭터 · 회갈색 단색 · 미니멀 도형 기반

## 1. 캐릭터 결정 근거

### 1.1 후보 비교 (의사결정 기록)

| 옵션 | 인상 | 메시지 정합 | 글로벌 인지 | 결정 |
|---|---|---|---|:---:|
| 판다 | 귀엽지만 게으름 | "푹 쉬어요" → 충돌 | 중국 한정 | ✗ |
| 쿠앤 | 호주 한정 | 졸린 이미지 → 충돌 | 낮음 | ✗ |
| **올빼미** | 학습·지혜 | "각성·주목" → 정합 | 보편적 | **✓** |
| 추상 도형 | 차가움 | 정서적 연결 약함 | — | ✗ |

### 1.2 올빼미 선택 이유

**(1) 학습·지혜의 원형적 상징**
한국·중국·서구 모든 권역에서 "학자·통찰·각성"을 상징. Phase 5 글로벌 확장 시 **재디자인 비용 없음**.

**(2) "각성·주목" 메시지와 정합**
집중 경고의 본질 = "깨어있게 만들기". 올빼미는 큰 눈 → 응시 → 주목으로 자연스럽게 연결. 졸린 동물(판다·코알라)은 메시지 충돌.

**(3) 표정 표현이 쉬움**
큰 눈 두 개로 1단계(걱정) → 2단계(놀람) → 3단계(또렷한 응시) 표현이 단순한 도형 변형으로 가능.

## 2. 이름

### 2.1 후보
- **"오우"** (Ow) — 한·중·영 발음 동일, 짧음
- **"후"** (Hoo) — 올빼미 울음소리 의성어, 영어권에서도 자연스러움
- "후니" / "오울이" — 한국어 친근감 더해진 버전 (선택적)

### 2.2 결정 (보류)
어흥 교수님 결정 사항. 본 문서에서는 **잠정적으로 "오우"** 사용. 추후 어흥 교수님이 확정 시 일괄 변경.

## 3. 시각 스타일

### 3.1 도형 기반 미니멀

```
   ╭───╮
  │ ◯ ◯ │     ← 큰 눈 두 개 (원)
   ╲ ▼ ╱      ← 부리 (작은 삼각형)
   │   │      ← 몸 (둥근 사각형)
   ╰───╯
```

구성:
- **머리·몸**: 둥근 사각형 (border-radius 50% 상단)
- **눈**: 두 개의 원
- **눈동자**: 더 작은 원 (검정)
- **부리**: 아래로 향한 작은 삼각형
- **귀깃 (선택적)**: 머리 위 두 개의 작은 삼각형

### 3.2 색상 (회갈색 단색)

```css
--mascot-base: #6B5B47;       /* 몸·머리 */
--mascot-light: #A89678;      /* 가슴·복부 라이트 영역 */
--mascot-eye-white: #F5EFE3;  /* 눈 흰자 */
--mascot-eye-pupil: #1A1A1A;  /* 눈동자 */
--mascot-beak: #D4923A;       /* 부리 (살짝 따뜻한 톤) */
```

**색상 선택 근거**:
- ElevenLabs 무채색 톤과 충돌 안 함
- Duolingo 부엉이의 채도 높은 그린과 차별화
- 어떤 배경(다크/라이트)에도 잘 어울림
- 눈만 강조되어 "응시" 메시지 살림

### 3.3 사이즈 가이드

| 사용 위치 | 크기 |
|---|---|
| 토스트 안 (1단계 경고) | 24-32px |
| 모달 안 (2단계 경고) | 80-120px |
| 풀스크린 모달 (3단계, demo CTA) | 160-240px |
| 학습자 온보딩 슬라이드 | 200-300px |
| 인터스티셜 퀴즈 헤더 | 32-48px |
| 학습자 마이페이지 인증서 | 64-96px |

## 4. 표정 변화 (3단계 + 추가)

### 4.1 1단계 — 걱정 (Concerned)
- 눈썹 살짝 위로 (V자 모양)
- 눈동자 약간 좌측으로
- 부리 살짝 작게
- 사용: 1단계 집중 경고 토스트

### 4.2 2단계 — 놀람 (Surprised)
- 눈동자 크게 (원의 80%)
- 눈동자 좌우로 미세 흔들림 애니메이션
- 부리 작은 O자
- 사용: 2단계 집중 경고 모달

### 4.3 3단계 — 응시 (Focused)
- 눈동자 정중앙 고정
- 눈썹 일직선 (단호함)
- 부리 닫힘
- 사용: 3단계 집중 경고 + 인터스티셜 퀴즈

### 4.4 격려 (Encouraging)
- 눈동자 정중앙
- 살짝 미소 (입꼬리 가상의 곡선)
- 사용: 정답 시, 학습 완료 카드

### 4.5 환영 (Welcoming)
- 눈동자 살짝 닫힘 (행복)
- 한쪽 날개 살짝 들기 (인사)
- 사용: demo CTA 모달, 학생 온보딩, 첫 로그인

### 4.6 졸림 (Sleepy)
- 눈 반쯤 감김
- 사용: ❌ 사용 금지 (학습 도구 메시지에 부적합)

## 5. 등장 위치 (제한적)

### 5.1 등장하는 곳
- 학습자 영상 시청 — **집중 경고 1·2·3단계**
- 학습자 영상 시청 — **인터스티셜 퀴즈 헤더**
- 학생 첫 사용 온보딩 (4슬라이드)
- 학습자 마이페이지 (`/profile`) — 스트릭·인증서
- 데모 페이지 — 체험 후 CTA 모달

### 5.2 등장하지 않는 곳
- 메인 사이트 (랜딩, features, pricing)
- 교수자 화면 전체 (대시보드, studio 등)
- 법적 문서 페이지 (legal, security)
- 베타 신청 폼·기관 견적 폼

**왜?**: 마스코트는 학습자(특히 학생)와의 정서적 연결을 위한 도구. 교수자·학과장에게는 "전문 도구"라는 인식이 더 중요하므로 등장 안 함.

## 6. 모션 가이드

### 6.1 기본 호흡 애니메이션 (Idle)
```css
.mascot-idle {
  animation: breathe 4s ease-in-out infinite;
}

@keyframes breathe {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.02); }
}
```

매우 미세한 호흡. 정적이지 않다는 인상만.

### 6.2 표정 전환
```css
.mascot-eye {
  transition: transform 300ms ease-out;
}

/* 1단계 → 2단계 변화 */
.mascot.attention-2 .mascot-eye {
  transform: scale(1.3);
}
```

### 6.3 등장 (Entrance)
```css
.mascot-entrance {
  animation: mascot-pop-in 600ms cubic-bezier(0.34, 1.56, 0.64, 1);
}

@keyframes mascot-pop-in {
  0% { transform: scale(0); opacity: 0; }
  100% { transform: scale(1); opacity: 1; }
}
```

### 6.4 흔들림 (놀람)
```css
.mascot-eye-pupil.surprised {
  animation: pupil-shake 1.5s ease-in-out infinite;
}

@keyframes pupil-shake {
  0%, 100% { transform: translateX(0); }
  25% { transform: translateX(-2px); }
  75% { transform: translateX(2px); }
}
```

### 6.5 인사 (Welcoming)
```css
.mascot-wing.waving {
```

## 7. 메시지 톤 (말풍선 콘텐츠)

마스코트가 말하는 메시지의 톤 가이드:

### 7.1 기본 원칙
- **친근하지만 학습 분위기 유지** (반말 X, 너무 격식 X)
- **짧게** (한 문장 권장, 두 문장 이내)
- **질문형 권유** (명령형 X)
- **학생을 다그치지 않음**

### 7.2 좋은 예
- ✅ "잠깐, 잘 따라오고 계신가요?"
- ✅ "오늘은 좀 피곤하신가봐요?"
- ✅ "잘 이해하고 계세요!"
- ✅ "이 부분 다시 볼까요?"

### 7.3 나쁜 예
- ❌ "집중하세요!" (명령)
- ❌ "왜 안 보고 있어요?" (질책)
- ❌ "어머, 또 자리 비우셨네요" (수동공격)
- ❌ "당신은 12분간 집중하지 않았습니다" (사무적)

### 7.4 다국어 톤

| 언어 | 톤 |
|---|---|
| 한국어 | 존댓말 + 친근 (-요) |
| 중국어 | 가벼운 친근 (你好, 我们一起 등) |
| 영어 | Casual but warm (Hey, let's, shall we 등) |

## 8. 사용 금지 사항

### 8.1 디자인적 금지
- ❌ 채도 높은 컬러 (그린·핑크 채도 강한 변형)
- ❌ 옷·소품 추가 (교수 모자, 안경 등 — 캐릭터 정체성 흐려짐)
- ❌ 사실적 일러스트 (도형 기반 유지)
- ❌ 애니메이션 캐릭터 스타일 (Pixar·Disney 스타일)
- ❌ 다른 동물과 함께 등장 (단일 캐릭터 유지)

### 8.2 사용 맥락 금지
- ❌ 광고 배너 (마스코트는 학습 컨텍스트 한정)
- ❌ 결제 페이지 (전문 도구 인상 유지)
- ❌ 에러 페이지 (에러는 차가운 톤이 더 적절)
- ❌ 부정행위 적발 알림 (위협적 인상 회피)

## 9. 구현 자산

### 9.1 SVG 컴포넌트
```jsx
// components/Mascot.jsx
export function Mascot({ expression = 'neutral', size = 120, ...props }) {
  // expression: 'neutral' | 'concerned' | 'surprised' | 'focused' | 'encouraging' | 'welcoming'
  return (
    <svg viewBox="0 0 200 240" width={size} {...props}>
      {/* 몸 */}
      <ellipse cx="100" cy="160" rx="70" ry="80" fill="var(--mascot-base)"/>
      {/* 가슴 라이트 */}
      <ellipse cx="100" cy="180" rx="40" ry="55" fill="var(--mascot-light)"/>
      {/* 눈 (표정에 따라 다름) */}
      <Eye expression={expression} side="left"/>
      <Eye expression={expression} side="right"/>
      {/* 부리 */}
      <polygon points="..." fill="var(--mascot-beak)"/>
    </svg>
  );
}
```

### 9.2 표정 prop으로 다양한 상태 표현
```jsx
<Mascot expression="welcoming" size={200} />  // demo CTA
<Mascot expression="focused" size={120} />    // 3단계 경고
<Mascot expression="encouraging" size={64} /> // 정답 시
```
